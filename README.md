# 🧮 Math Engine

Una librería matemática completa en Python que implementa desde cero métodos numéricos, álgebra lineal, cálculo, probabilidad, optimización, machine learning, teoría de grafos y mucho más.

## 📦 Módulos

| Módulo | Descripción |
|--------|-------------|
| `metodos_numericos` | Raíces, interpolación, integración numérica, EDOs |
| `algebra_lineal` | Matrices, sistemas lineales, espacios vectoriales |
| `calculo` | Límites, derivadas, integrales, cálculo vectorial |
| `probabilidad_estadistica` | Estadística descriptiva, distribuciones, inferencia, MCMC |
| `optimizacion` | Optimización 1D y multivariable (gradient descent, BFGS, evolutivos…) |
| `teoria_grafos` | BFS/DFS, caminos mínimos, flujos, coloración |
| `machine_learning` | Supervisado, no supervisado, redes neuronales, evaluación |
| `series_tiempo` | ARIMA, SARIMA, Holt-Winters, LSTM temporal |
| `procesamiento_senales` | FFT, filtros, wavelets, Kalman |
| `matematicas_discretas` | Combinatoria, teoría de números, lógica, grafos |
| `analisis_datos` | EDA, preprocesamiento, visualización |
| `simulacion` | Montecarlo, eventos discretos, modelos epidemiológicos |
| `finanzas_cuantitativas` | Black-Scholes, Markowitz, VAR, bonos |
| `procesamiento_lenguaje_natural` | TF-IDF, Word2Vec, LDA, análisis de sentimiento |
| `geometria_computacional` | Convex hull, Delaunay, Voronoi, Bézier |
| `criptografia` | AES, curvas elípticas, RSA, Diffie-Hellman |
| `metodos_cientificos` | Elementos finitos, lattice Boltzmann, caos, fractales |
| `optimizacion_combinatoria` | TSP, mochila, branch & bound, colonia de hormigas |

## 🚀 Instalación

```bash
git clone https://github.com/derekrex333/math_engine.git
cd math_engine
python -m venv .venv
.venv\Scripts\activate      # Windows
pip install -r requirements.txt
```

## 🧪 Tests

```bash
pytest tests/ -v
```
